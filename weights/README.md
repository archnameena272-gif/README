# YOLOv3 Weights

This folder is intended to store the pretrained YOLOv3 model weights.

## Download

Download the official YOLOv3 weights file:

**Filename:**

```
yolov3.weights
```

Place the downloaded file inside this folder:

```
weights/
└── yolov3.weights
```

## Note

The weights file is **not included** in this repository because it exceeds GitHub's file size limit.

Once the file is placed in this folder, you can run the `YOLOV3.ipynb` notebook without modifying the project structure.
